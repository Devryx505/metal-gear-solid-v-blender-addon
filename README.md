# import-metal-gear-solid-v

A Blender addon to import Metal Gear Solid V FMDL model files. 4.0 till 2.8 blender version

## What it does
Lets you pull FMDL models straight into Blender. Brings in the mesh, bones, normals, and UVs. Handles the coordinate system flip that Fox Engine uses so you don't have to worry about it.

## Features
- Import models with full skeleton support
- Custom normals preserved
- Multiple UV channels
- Bone bounding boxes (optional)

## Installation
1. Download the addon zip
2. Blender → Preferences → Add-ons → Install from File
3. Select the zip and enable it

## How to use
Hit N in the 3D viewport to open the side panel. Go to the Import tab and click "Import FMDL". Pick your model file and done.
